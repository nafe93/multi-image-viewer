# Multi Image Viewer

## English
Multi Image Viewer is a VS Code extension that allows you to open multiple folders with images and view identical images from different folders side by side by their index.

### Features
- Open multiple image folders
- Navigate through images by index
- Compare sets of images side by side

### Commands
- `Multi Image Viewer: Open Folders` — Select folders with images
- `Multi Image Viewer: Next Image` — Show the next set of images
- `Multi Image Viewer: Previous Image` — Show the previous set of images

---

## Русский
Multi Image Viewer — это расширение для VS Code, которое позволяет открывать несколько папок с изображениями и просматривать одинаковые изображения из разных папок рядом, по их индексу.

### Возможности
- Открытие нескольких папок с изображениями
- Навигация по изображениям через индекс
- Сравнение наборов изображений рядом

### Команды
- `Multi Image Viewer: Open Folders` — выбрать папки с изображениями  
- `Multi Image Viewer: Next Image` — следующий набор изображений  
- `Multi Image Viewer: Previous Image` — предыдущий набор изображений
